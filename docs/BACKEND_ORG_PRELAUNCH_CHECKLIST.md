## 发布门槛

测试环境必须连续通过：

```powershell
npm run db:migrate:test
npm run build:portals:test
npm run test:prelaunch
npm run check
```

正式上线前还必须在 Linux 测试环境完成 Nginx、PM2、HTTPS、备份恢复和微信沙箱/审核链路验证。当前状态适合进入测试环境部署，不适合直接向真实用户开放。
