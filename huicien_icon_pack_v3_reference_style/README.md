# 汇慈恩陪诊图标包 v3.0

本版按照用户提供的参考风格重新统一：

- 圆角线性图标
- 蓝色医疗服务主色
- 极浅白蓝圆角卡片
- 线宽统一、识别清楚
- 底部 Tab 同时提供灰色线性未选中态与蓝色面性选中态
- 医院不再使用汉字简称
- 服务不再使用“陪、检、药、报”等文字图标
- “我的”页面保持单一蓝白体系

共 67 枚业务图标，另含 5 枚 Tab 的选中与未选中状态。

## 视觉参数

- 图标网格：24 × 24
- 默认线宽：2.05px
- 主色：#1261D3
- 未选中灰：#697386
- 卡片背景：#FBFDFF
- 卡片边框：#DCE8F6
- 圆角卡片画布：72 × 72
- 卡片圆角：17px

## 输出目录

- `svg/outline/`：蓝色透明背景线性 SVG
- `svg/tile/`：圆角浅色卡片 SVG
- `svg/tab-unselected/`：Tab 灰色未选中态
- `svg/tab-selected/`：Tab 蓝色面性选中态
- `png/`：1x、2x、3x PNG
- `miniprogram/`：可直接复制到小程序的 PNG、组件和映射文件
- `preview-reference.png`：参考风格核心预览
- `preview-all.png`：全套图标预览

## 推荐使用

- 首页主功能：`tile`，约 96–108rpx
- 服务与医院卡片：`tile`，约 72–88rpx
- “我的”列表：`tile`，约 56–64rpx
- 普通信息行：`outline`，约 32–40rpx
- Tab 未选中：`tab-unselected`
- Tab 选中：`tab-selected`

运行目录只保留实际使用 PNG，SVG 放在设计资源目录，不要全部打入小程序主包。
